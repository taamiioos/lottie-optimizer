# Video to Lottie

Конвертирует видеофайл в Lottie JSON. Извлекает кадры из видео и упаковывает их в формат Lottie — каждый кадр становится отдельным ассетом и слоем анимации.

Полученный JSON можно сразу открыть в оптимизаторе — он найдёт покадровую последовательность и сожмёт её в H.264/MP4.

## Использование

1. Открыть `video-to-lottie.html`
2. Перетащить видеофайл или нажать для загрузки (поддерживаются форматы, которые воспроизводит браузер)
3. Настроить параметры
4. Нажать "Конвертировать"
5. Скачать `.lottie.json`

## Параметры

**FPS** — количество кадров в секунду.

**Макс. кадров** — ограничение на общее количество извлечённых кадров. Защита от случайного конвертирования длинного видео в тысячи кадров.

**Качество WebP** — качество сжатия при конвертации кадров в WebP (0.0–1.0). Значение 0.85 даёт хороший баланс между размером и качеством.

## Как работает

**Извлечение кадров через WebCodecs (основной метод)**

- MP4Box демультиплексирует видеоконтейнер и извлекает все закодированные сэмплы
- VideoDecoder декодирует сэмплы в сырые кадры
- Каждый кадр рисуется в OffscreenCanvas и конвертируется в WebP Blob
- Кадры отбираются по временным меткам согласно заданному FPS

**Извлечение кадров через `<video>` (fallback)**

- Видео загружается в скрытый `<video>` элемент
- Для каждого нужного момента времени устанавливается `currentTime`
- После события `seeked` кадр рисуется на Canvas
- Работает в любом браузере, но медленнее

**Сборка Lottie JSON**

- Каждый кадр кодируется в base64 data URL через FileReader
- Создаётся ассет с id `frame_N`, шириной и высотой видео
- Слой с типом `ty: 2` (изображение) показывает ровно один кадр: `ip: N`, `op: N+1`
- Итоговый JSON содержит столько слоёв сколько кадров, `fr` равен заданному FPS, `op` равно количеству кадров
