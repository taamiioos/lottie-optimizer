# Lottie Optimizer

Набор инструментов для оптимизации и воспроизведения Lottie-анимаций в браузере. 

## Инструменты

**Оптимизатор** (`demo/optimizer.html`) — принимает Lottie JSON с base64-изображениями внутри. Конвертирует картинки в WebP, находит покадровые последовательности и кодирует их в H.264/MP4. Возвращает облегчённый JSON и ZIP с ассетами.

**Плеер** (`demo/player.html`) — воспроизводит оптимизированную анимацию. Принимает JSON + ZIP, распаковывает ассеты, декодирует видеокадры и запускает анимацию через lottie-web.

**Video to Lottie** (`video-to-lottie/video-to-lottie.html`) — конвертирует видеофайл в Lottie JSON. Полезно для превращения видео в анимацию, которую можно затем оптимизировать.

## Быстрый старт

1. Открыть `demo/optimizer.html`
2. Перетащить `.json` файл с Lottie-анимацией
3. Нажать "Применить" и дождаться завершения
4. Скачать `optimized.json` и `assets.zip`
5. Открыть `demo/player.html`, загрузить оба файла, нажать "Воспроизвести"
