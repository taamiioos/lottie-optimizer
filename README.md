# Lottie Optimizer

## Возможности

- Конвертирует последовательности кадров в H.264 видео (WebCodecs + MP4Box)
- Сжимает изображения в WebP через Canvas
- Дедуплицирует одинаковые ассеты по SHA-256
- Упаковывает результат в JSON + ZIP
- Встроенный плеер с WebCodecs-декодированием и фоллбэком

**Вход:** Lottie JSON с base64-изображениями внутри

**Выход:** JSON со ссылками на файлы + ZIP-архив
